## Project Structure:
* FinalProject/
    * Citations - Contains ChatGPT logs from use in EDA
    * Data - Contains all of the data files for our project
        *  section_of_road/
            *  INSTRUCTIONS.txt - A file containing early data management notes
            *  readme.md - A link to the original data set
            *  section_of_road_on_map.pdf a file showing the analyzed section of road on google maps
        *  data sources ranked.md - An early dataset exploration and analysis
        *  full_station_data.csv/pkl - A csv and equivalent pickle file containing the aggregated dataset.
        *  data_preprocessing.py - Handles data management and preprocessing
        *  eda.ipynb/html/pdf - A workbook containing initial EDA
    * Results - Contains a few output images used in the report
    * Models - A folder containing the utilized models. Instructions for running the code are included.
